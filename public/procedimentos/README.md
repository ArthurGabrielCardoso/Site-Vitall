# Imagens de Procedimentos

Esta pasta contém todas as imagens dos procedimentos médicos do VitallCheck-Up.

## Estrutura recomendada:
```
procedimentos/
├── limpeza-facial.webp
├── microagulhamento.webp
├── peeling-quimico.webp
├── botox.webp
├── preenchimento.webp
└── ...
```

## Diretrizes:
- Formato: WebP (melhor compressão)
- Tamanho: 800x600px ou proporção similar
- Peso: máximo 150KB por imagem
- Nomes: kebab-case descritivo

## Otimização:
- Use ferramentas como TinyPNG ou Squoosh.app
- Mantenha qualidade entre 80-90%
